# 机审策略效果评估与优化项目（Excel 输入 → 多 sheet 报告 + 颜色标记）

## 项目简介
本项目模拟新闻内容场景下的规则型机审策略业务流程，围绕“规则执行效果是否准确、哪些关键词和内容类型容易产生错误、后续应如何调整策略”展开分析。

项目以一批新闻标题为样本，先依据预设规则进行机器审核，再结合人工复核结果，对机审效果进行统计分析、问题定位和优化建议输出，重点展示机审策略评估与迭代分析能力。


## 项目目标
本项目主要解决以下问题：

机审规则执行后的整体效果如何
不同识别内容、不同机审结果下的准确率表现如何
哪些关键词可能集中带来错误判定
哪些细分场景下机审正确率较低，值得进一步优化
如何基于分析结果提出策略迭代建议


## 业务规则示例
项目中以新闻所属国家作为一个基础分类维度，示例规则包括：

美国新闻：删除
中国新闻：不分发
其他国家新闻：通过

同时结合关键词识别和内容识别结果，对机审策略执行效果进行观察与分析。

说明：本项目为模拟业务场景，规则设计相对简化，重点不在构建复杂审核系统，而在展示机审策略效果分析与优化思路。

## 项目流程
- 1. 数据清洗
清理无效数据
判断哪些记录保留、哪些删除
统计有效样本量
- 2. 内容识别结果统计
对样本进行识别后，统计：
各国家新闻数量
各机审结果数量
未识别出具体国家的数量
不同国家在不同机审结果下的分布情况
- 3. 准确率分析
基于人工复核结果，计算：
不同国家分类下的准确率
不同机审结果分类下的准确率
国家 × 机审结果交叉场景下的准确率
- 4. 典型样本抽取
针对不同机审结果分类，分别抽取典型样本，用于：
观察规则表现
辅助解释数据结果
支持后续策略优化判断
- 5. 错误关键词定位
筛选人工复核为错误的数据，分析错误样本是否集中出现在某些关键词上，从而定位可能需要调整的关键词规则。
- 6. 分层分析
从以下维度进行进一步切分：
文本长度
是否包含英文
内容来源
观察各细分场景下的机审正确率差异，定位更具体的问题区间。
- 7. 输出迭代建议
基于整体分析结果，给出机审策略优化建议，包括：
哪些关键词需要收严或放宽
哪些细分场景需要单独处理
哪些规则表现稳定可继续保留
